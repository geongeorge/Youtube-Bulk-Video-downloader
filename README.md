# Youtube multi downloader - node.js

Download multiple youtube videos easily.

- Recommended Node version 16.11.0

## Install ⚙️

```
npm install
```

## Input file

Create `videos.json` with input files following the structure:

```json
{
  "list":[
    "https://youtu.be/dQw4w9WgXcQ"
  ]
}
```

## Run 🚀
```
node app.js
```
